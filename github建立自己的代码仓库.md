### 建立自己的代码仓库

github的注册不必赘述

1. 首先在hub新建一个自己的仓库，如果是新注册的账户会自动提示你创建
2. git init
3. git remote add origin https://github.com/zyl-fun/blog.git（这是你自己的远程仓库的地址）
4. git add .
5. git commit -m '写上自己的提交说明'
6. git push -u origin master
7. git pull 从远端服务器拉取代码
